# Community Bridge Documentation

**Universal Translation Layer for FiveM** — One API for every framework, inventory, notification system, and more.

## What is Community Bridge?

Community Bridge is a universal compatibility layer for FiveM that auto-detects your installed resources and provides a single, consistent API across all of them. Write your scripts once and they work everywhere — ESX, QBCore, QBX, or standalone.
