# Weather
The Weather module provides client-side weather sync control.

## Client Functions

### GetResourceName

Returns the weather resource name.

```lua
local name = Bridge.Weather.GetResourceName()
-- Returns: string (always "default")
```

**Description:**
Returns the name of the Weather module resource. Used for internal identification. Always returns `"default"` for the built-in Weather module.

### ToggleSync

Toggles the weather sync system on or off.

```lua
Bridge.Weather.ToggleSync(toggle)
```

| Parameter | Type | Description |
|-----------|------|-------------|
| toggle | boolean | `true` to enable sync, `false` to disable |
