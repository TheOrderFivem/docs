## Framework Fallback

If none of the supported inventory resources are detected, Community Bridge falls back to the **framework's built-in inventory functions** (`Framework.AddItem`, `Framework.RemoveItem`, etc.). This means item operations will still work on any ESX or QB-based server even without a dedicated inventory resource.
