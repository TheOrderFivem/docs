# HelpText — Client

## ShowHelpText

Displays a help text notification on screen.

```lua
Bridge.HelpText.ShowHelpText(text)
```

| Parameter | Type | Description |
|-----------|------|-------------|
| text | string | Text to display |

## HideHelpText

Hides the currently displayed help text.

```lua
Bridge.HelpText.HideHelpText()
```
