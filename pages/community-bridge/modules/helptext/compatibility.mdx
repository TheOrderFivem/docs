# HelpText — Compatibility

## Supported Resources

| Resource | Notes |
|----------|-------|
| `cd_drawtextui` | Full support |
| `jg-textui` | Full support |
| `lab-HintUI` | Full support |
| `lation_ui` | Full support |
| `okokTextUI` | Full support |
| `ox_lib` | Full support |
| `zsxui` | Full support |

## Configuration

Set `BridgeSharedConfig.HelpText` to `"auto"` for auto-detection or specify a resource name directly.

If no supported resource is found, Community Bridge falls back to the framework's notification system.
