# Markers — Server

## Create

Registers a marker that will be synced to clients.

```lua
Bridge.Marker.Create(options)
```

The options are the same as the client version.

## Remove

Removes a server-side marker.

```lua
Bridge.Marker.Remove(id)
```

| Parameter | Type | Description |
|-----------|------|-------------|
| id | string | Marker ID |
